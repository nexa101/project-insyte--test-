## 🥬 스타라팜 셀러 도구

농수산물 스마트스토어 셀러를 위한 웹 도구입니다. 별도 설치나 로그인 없이 휴대폰·PC 브라우저에서 바로 사용할 수 있습니다.

**👉 사용하기: https://nexa101.github.io/project-insyte--test-/starafarm/**

### 1. 마진 계산기

- 플랫폼 선택: 스마트스토어(6.0%) · 쿠팡(11.7%) · 토스(11.8%) · G마켓(13.0%) · 11번가(13.0%) · 직접입력
- 수수료율은 언제든 직접 수정 가능 (플랫폼 정책 변경 대응)
- 공급가 · 택배비 · 포장비 · 광고비 · 판매가를 입력하면 즉시 계산:
  - 플랫폼 수수료, 총원가, 순이익, 마진율
  - 목표 마진율 기준 권장 판매가
  - 희망 할인율 기준 추천 정상가 (쿠팡·토스·G마켓 할인가 노출용)
- 플랫폼별 수수료·순이익·마진율 비교표
- 역마진 경고, 목표 마진 미달 시 권장 판매가 안내
- 입력값은 기기에 자동 저장 (서버 전송 없음)

### 2. 상품명 생성기

- 상품 · 산지 · 품종 · 중량 · 브랜드 + 강조 키워드(산지직송, 못난이, 선물용 등)를 조합해 상품명 후보 생성
- 글자 수 표시, 스마트스토어 권장 50자 초과 시 경고
- 같은 단어 반복 자동 제거, 한 번 클릭으로 복사

### 3. 리뷰 답글 생성기

- 고객 리뷰를 붙여넣으면 리뷰 유형(긍정/아쉬움/불만)과 언급 주제(맛·신선도·배송·포장 등)를 자동 감지
- 상황에 맞는 답글 후보 3개를 생성, 누를 때마다 다른 조합
- 불만 리뷰는 사과 → 개선 약속 → 문의 안내 구조 (교환·보상 확답 없이 안전한 톤)

### 계산 공식

```
수수료        = 판매가 × 수수료율
총원가        = 공급가 + 택배비 + 포장비 + 광고비 + 수수료
순이익        = 판매가 − 총원가
마진율        = 순이익 ÷ 판매가 × 100
권장 판매가   = (공급가 + 택배비 + 포장비 + 광고비) ÷ (1 − 수수료율 − 목표마진율)
추천 정상가   = 판매가 ÷ (1 − 희망할인율)   ※ 100원 단위 올림
```

### 기술

- 순수 HTML/CSS/JavaScript 단일 파일 (`starafarm/index.html`) — 외부 라이브러리, 서버, API 없음
- 모바일 우선 반응형, 라이트/다크 테마 지원
- 수수료율은 플랫폼 정책에 따라 변할 수 있으므로 실제 정산 내역과 비교 후 수수료율 칸에서 직접 수정해 사용
